Все задания хранятся в папке db. В ней находится файл all_tasks.txt, а также столько папок, сколько существует заданий а данный момент.

Файл all_tasks.txt представляет собой список всех существующих заданий. На каждой строчке файла представлена информация для конкретного задания. Информация состоит из:
- Названия задания;
- Сложности задания;
- Выполненности задания (1 если выполнено, 0 - если не выполнено).
Информация на каждой строчке разделена пробелами.

Для каждой строки в файле all_tasks.txt, описывающей определенное задание, в папке db должна быть определена папка с названием, совпадающим с названием задания, записанным в строке информации.
Эти папки нужны для полного описания задач каждого задания. В каждой папке должно храниться минимум три файла:
- Файл preview.png - картинка, используемая в меню выбора для предпросмотра всего задания;
- Файл task.txt - текстовый файл, описывающий каждую задачу задания;
- Файлы *1.png, *2.png, ..., *N.png - картинки, на которых изображены отдельные задачи. Файлов с картинками должно быть столько, сколько задач описано в файле task.txt.

Файл task.txt описывает задание следующим образом: так как задание это набор задач, выполняемых последовательно, файл описывает каждую задачу и соответствующий ей набор фигур последовательно. Для описания каждой задачи нужно записывать в файл task.txt определенные строки по следующему алгоритму:
1) Для того, чтобы начать описание задачи, нужно написать название картинки *i.png, описывающей эту задачу;
2) На следующих строках (минимум на одной) должно располагаться описание фигур, на которые нужно наступить. Сколько фигур - столько должно быть строк, описывающих фигуры. 
Для описания фигуры нужно выполнить следующие шаги:
	а) Написать название фигуры (названия ВСЕХ фигур в задании должны отличаться). Это может быть осмысленное название или просто порядковый номер. После названия фигуры нужно поставить двоеточие и пробел.
	б) Нужно задать координаты центра фигуры в пикселях, учитывая, что верхний левый пиксель имеет координаты 1, 1, а нижний правый пиксель имеет координаты 1920, 1080. Координаты нужно задавать в формате "x, y", после координат нужно поставить точку с запятой и пробел.
	в) Нужно задать радиусы фигуры. Каждая фигура, изображаемая на картинках задач, представляется эллипсом. У эллипса есть два радиуса (большая и малая полуоси). Нужн определить количества пикселей в радиусах и записать в строку в формате "rx, ry", после радиусов нужно поставить точку с запятой и пробел.
	г) Нужно задать угол поворота эллипса от 0 до 359, где 0 указывает на то, что эллипс не нужно поворачивать.
	В конце концов, описание одной фигуры дожно выглядеть следующим образом:
		<название>: x, y; rx, ry; <угол>
3) После того, как были описаны все фигуры задачи, нужно завершить описание текущей задачи, написав слово "end" на отдельной строке. После этого можно со следующей строки начать описание следующей задачи или больше ничего не писать, если законченная задача была последней.
