Все задания хранятся в папке db. В ней находится файл all_tasks.json, а также столько папок, сколько существует заданий а данный момент.

Файл all_tasks.json представляет собой список всех существующих заданий. Файл выглядит следующим образом:
{
	"names": [
		"<Название первого задания>",
		"<Название второго задания>",
		...
		"<Название последнего задания>"
	]
}

Для каждого названия задания в файле all_tasks.json, в папке db должна быть определена папка с названием, совпадающим с названием задания.
Эти папки нужны для полного описания задач каждого задания. В каждой папке должно храниться минимум 3 файла:
- Файл preview.png - картинка, используемая в меню выбора для предпросмотра всего задания;
- Файл task.json - файл, хранящий параметры задания, а также описывающий каждую задачу задания;
- Файлы *1.png, *2.png, ..., *N.png - картинки, на которых изображены отдельные задачи. Файлов с картинками должно быть столько, сколько задач описано в файле task.json.

Файл task.json выглядит следующим образом:
{
	"is_complete": <true если задание было ранее выполнено, false - иначе>,
	"difficulty": <Уровень сложности>,
	"all_exes": []
}
all_exes - массив, каждый элемент которого описывает конкретную задачу задания. Каждый элемент массива выглядит следующим образом:
{
	"img": "<Название файла с картинкой задания>",
	"ex_figs": []
}
ex_figs - массив, каждый элемент которого описывает конкретную фигуру задачи. Каждая задача может содержать несколько фигур, но не менее 1. Каждый элемент массива ex_figs выглядит следующим образом:
{
	"name": "<Название фигуры>",
	"center": [cx, cy],
	"radius": [rx, ry],
	"angle": a
}
Каждая фигура описывается эллипсом с помощью центра эллипса, его радиусов (большой и малой полуосей) и угла наклона. center - центр фигуры в пикселях, radius - радиус фигуры в пикселях, angle - угол фигуры в градусах от 0 до 359.
